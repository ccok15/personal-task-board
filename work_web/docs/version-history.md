# 版本记录

当前稳定开发版本：`v1.1`

## v1.1

这一版作为当前网站的第一个明确版本基线，主要目的是把现阶段已经定下来的产品形态固定下来，方便后续继续小步迭代。

### 当前版本范围

- 首页作为主工作台
- 游客只读任务管理页
- 管理员任务管理与编辑能力
- 浅色 / 深色主题切换
- 本地 `launchd` 常驻开发服务
- 工作站域名接入方案

### 当前页面特征

- 页面头部展示版本号 `v1.1`
- 首页任务卡默认压缩展示，点击后展开
- 首页支持按状态筛选当前任务
- 任务管理支持搜索
- 管理员与游客的可见范围和操作能力已区分

### 版本使用约定

- 页面展示版本号统一从这里读取：
  - `/Users/lsl/new_gpt/web_lsl/work_web/lib/version.ts:1`
- `package.json` 的版本号与页面主版本保持一致：
  - 页面展示：`v1.1`
  - 包版本：`1.1.0`

### 后续升级建议

- 小功能修正但不改变整体结构：继续沿用 `1.1.x`
- 一轮明确的新功能收敛后：升级到 `1.2`
- 只有产品结构或定位明显变化时，再进入 `2.0`
