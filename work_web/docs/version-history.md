# 版本记录

当前稳定开发版本：`v1.2`

## v1.2

这一版是在 `v1.1` 基础上的一次功能增强版本，重点补齐了首页工作流里的“进行中任务摘要”和真实进度推进体验，同时继续把交互压缩得更紧凑。

### 当前版本范围

- 首页进行中任务摘要栏
- 进行中任务真实进度条
- 管理员可直接在摘要栏推进任务进度
- 长按直达完成并自动流转到已完成
- 任务进度持久化保留
- 悬浮球新建任务入口
- 页面版本号升级为 `v1.2`

### 本次新增与调整

- 首页顶部新增“进行中任务”摘要栏，只聚焦展示进行中的任务
- 摘要栏中的任务支持真实进度条，按 `10%` 为一档持久化保存
- 管理员可在摘要栏中：
  - 点击左半边减少 `10%`
  - 点击右半边增加 `10%`
  - 长按右半边 `0.8` 秒直接完成
- 任务进度会在 `待处理 / 阻塞 / 搁置 / 进行中` 之间保留
- 已完成任务重新回到未完成状态时，进度会回退到可继续推进的状态
- 首页新建任务入口改成悬浮球 + 弹窗
- 首页与任务管理页顶部大标题区进一步收紧
- “进行中任务”摘要条样式调整为更紧凑的条状视觉，并补了青蓝色任务流动效

### 当前页面特征

- 页面头部展示版本号 `v1.2`
- 首页顶部优先显示“进行中任务”摘要
- 首页完整任务卡仍保留展开查看和状态流转能力
- 任务管理支持搜索
- 管理员与游客的可见范围和操作能力已区分

## v1.1

这一版作为当前网站的第一个明确版本基线，主要目的是把现阶段已经定下来的产品形态固定下来，方便后续继续小步迭代。

### 当前版本范围

- 首页作为主工作台
- 游客只读任务管理页
- 管理员任务管理与编辑能力
- 浅色 / 深色主题切换
- 本地 `launchd` 常驻开发服务
- 工作站域名接入方案

### 当前页面特征

- 页面头部展示版本号 `v1.1`
- 首页任务卡默认压缩展示，点击后展开
- 首页支持按状态筛选当前任务
- 任务管理支持搜索
- 管理员与游客的可见范围和操作能力已区分

### 版本使用约定

- 页面展示版本号统一从这里读取：
  - `/Users/lsl/new_gpt/web_lsl/work_web/lib/version.ts:1`
- `package.json` 的版本号与页面主版本保持一致：
  - 页面展示：`v1.2`
  - 包版本：`1.2.0`

### 后续升级建议

- 小功能修正但不改变整体结构：继续沿用 `1.2.x`
- 一轮明确的新功能收敛后：升级到 `1.3`
- 只有产品结构或定位明显变化时，再进入 `2.0`
